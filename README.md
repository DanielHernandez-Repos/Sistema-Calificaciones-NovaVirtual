# Sistema-Calificaciones-NovaVirtual
Sistema de gestión de calificaciones de los alumnos de modelo virtual de NovaUniversitas.

1-. Cada que se suba algo poner el comando
git add .
git commit -m "Se agregó registro de alumnos"

2-. Luego 
git push

3.- Antes de empexar a trabajar en el dia

3.1 .- cambiar al main
git checkout main

3.2 .- Descargar cambios recientes:
git pull origin main

3.4 .- Volver a tu rama:
git checkout daniel

3.5 .- Actualizar tu rama:
git merge main

4-. Integrar trabajo terminado: supongamos que uno termino el su proyecto completo, hacer en git 
Pull Requests
↓
New Pull Request
↓
daniel → main
GitHub mostrará los cambios.

5-. Edgar revisa y si todo está bien:
Merge Pull Request
Ahora esos cambios pasan a main.
